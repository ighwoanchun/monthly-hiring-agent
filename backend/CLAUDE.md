# 월간 채용 분석 에이전트

이 프로젝트는 채용 플랫폼의 월간 데이터를 분석하여 리포트를 생성합니다.

## 에이전트 역할

- 엑셀 파일(지표분석용_sheet.xlsx)을 읽고 분석
- Part A(실적)/Part B(파이프라인) 구조의 마크다운 리포트 생성
- Confluence 업로드 및 Slack 알림 (MCP 연동 시)

## 데이터 구조

| 시트명 | 기준 | 용도 |
|--------|------|------|
| 월통합분석 | 월별 집계 | KPI 추이, 매출 |
| 합격기준리드타임_raw | **합격월** 기준 | Part A 실적 분석 |
| 지원기준리드타임_raw | **지원월** 기준 | Part B 파이프라인 분석 |

## 용어 정의 (필수)

| 컬럼명 | 표기 | 의미 |
|--------|------|------|
| pass_cnt | **서류통과 수** | 서류 심사 통과자 (⚠️ 합격 아님!) |
| hire_cnt | **합격 수** | 최종 합격자 |
| new_com_accept | **신규기업 가입** | 실제 가입 완료 기업 |
| recruit_fee | 수수료 매출 | 합격 시 수수료 |
| flat_rate_fee | 정액제 매출 | 월정액 서비스 |
| ad_sales | 광고 매출 | 광고 상품 |

## 전환율 참조값

**지원→합격:** 당월 11.3% / 전월 **45.2%**⭐ / 전전월 25.8% / 전전전월 17.7%
**서류통과→합격:** 당월 26% / 전월 **37.5%**⭐ / 전전월 14% / 전전전월 7.5%

## 리포트 구조

### Part A. 실적 분석 (합격기준)
1. Executive Summary - 6개 지표 + 테이블형태
2. 월별 KPI 추이 - 4개월 + 열에 전월대비 증감
3. 매출 구조 - 수수료/정액제/광고 + 테이블형태
4. 건당 수수료 단가 추이 - 수수료매출/합격수/건당단가 + 테이블형태
5. 합격자 분석 
5-1. 직군별/합격수/비중/전월/전월대비증감%/상태/리드타임
5-2. 기업규모별
5. 성과 평가

### Part B. 파이프라인 분석 (지원기준)
6. 지원 현황 - 직군별/기업규모별
7. 퍼널 전환 분석 ⭐핵심
8. 익월 예측

### 공통
9. 리스크 & 기회
10. 액션 아이템

## 분석 규칙

- MoM = (당월-전월)/전월×100
- 상태: >+5% → 📈, <-5% → 📉, 그 외 → ➡️
- 리드타임 가중평균 = Σ(리드타임×합격수)/Σ합격수
- **직군별/기업규모별 합계 = hire_cnt 일치 필수**
- TOP 10 외 → "기타", null → "미분류"
